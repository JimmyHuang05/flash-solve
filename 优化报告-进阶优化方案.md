## P0 — 立即执行

### 1. 提取内联 SVG 为 Icons 组件

**现状：** 全项目 10 个文件中散布 33 个内联 `<svg>`，相同图标多处重复。

**重复图标统计：**

| 图标 | 出现次数 | 文件分布 |
|------|:--------:|---------|
| 关闭 / X | 2 | BaseModal, UploadPage |
| 返回箭头 | 2 | HistoryModal, WorkbookPage |
| 设置齿轮（完全相同） | 2 | Navbar, SettingsModal |
| 加号 | 2 | SquarePage, SolveResult |
| 勾选 check | 2+ | VipPage, HistoryModal |
| 上/下箭头 | 2 | WorkbookPage |
| 其他唯一图标 | ~20 | 各处 |

**方案：** 新建 `src/components/Icons.jsx`，将每个 SVG 封装为 React 组件：

```jsx
// Icons.jsx
export function IconClose({ size = 18, className }) {
  return (
    <svg xmlns="http://www.w3.org/2000/svg" width={size} height={size} viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2.5" strokeLinecap="round" strokeLinejoin="round" className={className}>
      <path d="M18 6 6 18"/><path d="m6 6 12 12"/>
    </svg>
  )
}
// ... 其余图标同理
```

**预期效果：** 减少 ~60% SVG 重复代码，更换图标风格只需改一处。

### 2. 在 index.css 中添加复用类

**现状：** 以下样式模式在全项目重复出现 5-13 次：

| 模式 | 出现次数 | 示例 |
|------|:--------:|------|
| 卡片容器 `.card` | 7 | `bg-white/70 backdrop-blur-lg rounded-[1.5rem] border border-white/90 shadow-glass` |
| 主渐变按钮 `.btn-primary` | 13 | `bg-gradient-to-r from-warm-coral to-warm-coralHover text-white font-bold rounded-xl shadow-md ...` |
| 详情行 `.field-row` | 13 | `px-4 py-3 bg-white/60 rounded-xl border border-stone-100/80` |
| 答案框 `.answer-box` | 3 | `px-4 py-3 bg-green-50/80 rounded-xl border border-green-100/60` |
| 解析框 `.explanation-box` | 3 | `px-4 py-3 bg-stone-50 rounded-xl border border-stone-100` |
| 标签徽章 `.badge` | 8+ | 多种变体 |

**方案：** 在 `index.css` 中添加：

```css
@layer components {
  .card {
    @apply bg-white/70 backdrop-blur-lg rounded-[1.5rem] border border-white/90 shadow-glass;
  }
  .card-hover {
    @apply card hover:bg-white/90 hover:shadow-glass-lg;
  }
  .btn-primary {
    @apply bg-gradient-to-r from-warm-coral to-warm-coralHover text-white font-bold rounded-xl shadow-md shadow-warm-coral/20 hover:shadow-lg transition-all active:scale-95;
  }
  .field-row {
    @apply px-4 py-3 bg-white/60 rounded-xl border border-stone-100/80;
  }
  .badge {
    @apply text-[10px] font-medium text-stone-400 bg-white/80 border border-stone-200 px-2 py-0.5 rounded-full;
  }
}
```

**预期效果：** 减少 ~40% Tailwind 类名重复，设计风格修改只需改 CSS 一处。

### 3. 提取 SegmentedControl 组件

**现状：** 4 处使用了相同的"选中/未选中"双态按钮组：

| 位置 | 用途 |
|------|------|
| HistoryModal | 解题记录 / 题生题记录 Tab |

未选中态：`text-stone-400 bg-white/80 border border-stone-200 hover:border-warm-coral/30`

**方案：** 提取为 `<SegmentedControl options={[]} value={} onChange={} />` 组件。

### 4. 消除重复的 auth session 请求

**现状：** `SquarePage.jsx`（第 15 行）和 `UploadPage.jsx`（第 46 行）各自重复调用 `supabase.auth.getSession()`，而 `App.jsx` 已经获取了 session 并通过 props 传递。

**方案：** 移除这两个组件内部的 `getSession()` 调用，改为从 props 接收 `session`：

```diff
-  const [session, setSession] = useState(null)
-  useEffect(() => { supabase.auth.getSession().then(({ data }) => setSession(data.session)) }, [])
+  // session 从 props 接收（App.jsx 已传递）
```

**`SquarePage` 额外注意：** 当前 `SquarePage` 没有从 `App.jsx` 接收 `session` props，需要添加。
